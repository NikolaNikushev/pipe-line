A Sandcastle Documented Class Library
Flowly Namespace
This namespace contains all the classes that the Flowly Windows forms uses. Inside there are brief descriptions of each class and their methods.
Classes
  ClassDescription
Change
Class "Change" is meant to be used for the changes that will be made during runtime. For each activity of the user new "Change" will be created. So the "Undo" option will be available to the user.
ComponentDrawn
ComponentDrawn class is meant for the components that the program will have. Components as pipe, sink, merger and so on.
ConnectionPoint
"ConnectionPoint" is the place where each "Component" is connected with another one.
DialogWindow
Meant to be used for Dialog Windows that will be opened during runtime.
Form1
The main window form of the application.
Grid
Objects of this class will be the "drawing" place of the user. A grid will contain components in it.
Merger
Sub class of the "ComponentDrawn" class. A particular component called - Merger. Nothing specific in it. Set diffCurrFlowPossible needs to be false.
NamespaceDoc
Pipe
Sub class of the "ComponentDrawn" class. A particular component called - Pipe. Nothing specific in it. Set diffCurrFlowPossible needs to be false.
Pump
Sub class of the "ComponentDrawn" class. A particular component called - Pump. Nothing specific in it. Set diffCurrFlowPossible needs to be false.
Sink
Sub class of the "ComponentDrawn" class. A particular component called - Sink. Nothing specific in it. Set diffCurrFlowPossible needs to be false.
Splitter
Sub class of the "ComponentDrawn" class. A particular component called - Splitter. SPECIFICATION: IF NORMAL SPLITTER THEN setCurrFlowPossible attribute needs to be false. IF ADJUSTABLE SPLITTER THEN setCurrFlowPossible needs to be true. Because through the two outputs of the adjustable splitter different amounts of liquid/gas can go through.
SystemFlowly
Class named "SystemFlowly". Meant to possess many of the methods to manipulate, create, delete "ComponentDrawn"-s. Has a list for "Change"-s to keep track of them. Has a list meant to be a toolbox.
ToolboxComponent
Not a "ComponentDrawn"! This is a Component from the Toolbox! Can be just used to CREATE an instance of "ComponentDrawn"!

IEditable
This is an interface that enables an object to be edited

Direction
This is an emurations containg the direction a component can face Left or right. It is used to better display the value of the direction
PropertiesEnum
Enumeration that contains the type of propertie that can be set, if it is possible you can adjust the flow that can pass. If the component is an adjustable splitter it has the property of Adjustable. Properties: FLOW and Adjustable
