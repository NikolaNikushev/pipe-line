Splitter Class
Sub class of the "ComponentDrawn" class. A particular component called - Splitter. SPECIFICATION: IF NORMAL SPLITTER THEN setCurrFlowPossible attribute needs to be false. IF ADJUSTABLE SPLITTER THEN setCurrFlowPossible needs to be true. Because through the two outputs of the adjustable splitter different amounts of liquid/gas can go through.
